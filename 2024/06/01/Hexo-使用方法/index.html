Hexo 使用方法

Hexo 的安装

环境配置

电脑中已有 Git 和 Node.js 的可以跳过环境配置部分。

下载并安装 Git

在下面链接的网站中选择你的电脑系统的对应版本 Git 下载。

https://git-scm.com/download/

下载完成后，运行下载的安装程序进行安装。

下载并安装 Node.js

在下面链接的网站中下载 Node.js。

https://nodejs.org/zh-cn/

下载完成后，运行下载的安装程序进行安装。

为 Node.js 配置环境变量

这里只给出 Windows 的环境变量设置方法。打开设置，在设置中搜索“环境变量”，选择“编辑系统环境变量”，在弹出的窗口中点击按钮”环境变量“，在新窗口中的”系统变量“中找到名为”PATH“的项，双击打开，在打开的窗口中选择”新建“，然后输入 Node.js 的安装路径，完成后点击确定。

配置完成后，在命令行运行node -v，若出现了版本号，说明配置成功。

安装 Hexo

https://hexo.io/zh-cn/index.html

在你想要放置项目文件的目录下，右键，点击”Git Bash Here”，在打开的 Git 命令行窗口中，依次输入下面 5 条指令。最后一条指令是启动网页进程，如果成功打开了网页，说明安装成功。
